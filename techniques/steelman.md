# Steelman

## The Move
Before attacking any argument, position, or proposal:

1. State the strongest version the proponent would endorse
2. Add supporting evidence they may have omitted
3. Fix any logical gaps in their reasoning
4. THEN critique this strengthened version

If your critique survives the steelman, it's a real objection. If it only works against the weak version, it's not worth raising.

## When to Use
- Before any DA (devil's advocate) pass
- When evaluating proposals you disagree with
- During committee reviews where conformity bias is a risk
- When the user presents a position and asks for feedback

## Allergy
Critiquing weak versions of arguments to feel clever. Dismissing positions without understanding them.
